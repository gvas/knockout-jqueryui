Documentation
-------------
[http://gvas.github.com/knockout-jqueryui/](http://gvas.github.com/knockout-jqueryui/)

Build
-----
    git clone git://github.com/gvas/knockout-jqueryui.git knockout-jqueryui
    cd knockout-jqueryui
    npm install -g grunt-cli
    npm install
    grunt
